# sim

 A time-driven simulation framework.

时间驱动的仿真框架.

# 基本概念.

## 实体

实体是对显示中的实际物体的抽象.

在仿真过程中，实体的基本接口如下：
* 步进 (step)：

    实体根据当前状态，前进一步（按照规定时长）.
    
* 交互 (access)：

    实体根据场景中其他实体的状态，更新自己的状态.

## 场景

场景主要功能包括以下几点：

* 管理实体
  
  包括：增加实体、删除实体等.

* 驱动场景运行
  
  包括：管理实体步进、管理实体之间的相互交互.
